# DDNS

## 一键DDNS（基于ddns-go)

自动判断CPU架构，并下载最新版本二进制文件

## 一键脚本命令：

```
wget -qO- https://raw.githubusercontent.com/FrankLiangCN/DDNS/main/ddns.sh | bash
```

安装完成后请打开 http://IP:9876 端口进行配置
